# 5.env_var_config

*Created on:* 2026-01-03

## Description

Adding the possibility to load fast api settings and other parameters from env variable 

## Prerequisites

- Have python 3 installed

## Other change from previous stage

- adding `settings.py` in order to load app settings from env var
- adding the possibility to load from .env file 
- adding some validator to assert variable values

## Usage

- Set env variable (from example.env for quick test):
```
cp example.env .enc
source .env
```

- Deploy application using virtualenv :
```
python3 -m venv venv
source venv/bin/activate
python -m pip install -r requirements.txt
uvicorn app.main:app --host $HOST --port $PORT --reload
```

- Run test (in another terminal) :
```
python3 -m venv venv
source venv/bin/activate
python -m pytest tests/
```
