# api 

## Description 

Create a simple REST api using fastapi for TODO list project :
- Running api using python virtual environment and fastapi command (`fastapi dev app/main.py`)
- Using in memory storage for TODOs
- Using a simple bash script to easily test it
- Adding a .gitignore file to remove unused file in repository

## Prerequisites

- Have python 3 installed

## Other change from previous stage 

- no previous stage

## Usage

- Deploy application using virtualenv :
```
python3 -m venv venv
source venv/bin/activate
python -m pip install -r requirements.txt
fastapi dev app/main.py
```

- Open a new terminal and run bash test script :
```
