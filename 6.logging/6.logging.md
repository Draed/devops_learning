## Description

Adding logging capability

## Prerequisites

- Have python 3 installed

## Other change from previous stage

- add `logger.py` to configure logging for the app
- adding debug var for easily configure 
- using loguru package for later logging advanced usage

## Usage

- Set env variable (from example.env for quick test):
```
cp example.env .enc
source .env
```

- Deploy application using virtualenv :
```
python3 -m venv venv
source venv/bin/activate
python -m pip install -r requirements.txt
uvicorn app.main:app --host $HOST --port $PORT --reload
```

- Run test (in another terminal) :
```
python3 -m venv venv
source venv/bin/activate
python -m pytest tests/
```
