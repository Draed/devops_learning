# 4.python_docstring

*Created on:* 2025-12-28

## Description

Adding some code documentation based on python docstring.

## Prerequisites

- Have python 3 installed

## Other change from previous stage

- adding docstring from google style
- adding python 3.5 return type function syntax "->"
- rebder code documentation using pdoc

## Usage

- Deploy application using virtualenv :
```
python3 -m venv venv
source venv/bin/activate
python -m pip install -r requirements.txt
uvicorn app.main:app --host localhost --port 8000 --reload
```

- Run test (in another terminal) :
```
python3 -m venv venv
source venv/bin/activate
python -m pytest tests/
```

- Generate documentation using pdoc :
```
python3 -m venv venv
source venv/bin/activate
python -m pip install pdoc
pdoc app/ --output-dir docs --docformat google
```
