# 3.ci

*Created on:* 2025-12-23

## Description 

Adding CI (Continuous Integration)
- Adding github action workflow

## Prerequisites

- Have python 3 installed
- Enabling github pages in repository setting :
    - Navigate to your repository on GitHub.
    - Go to the "Settings" tab.
    - In the left sidebar, click on "Pages" under the "Code and automation" section.
    - Under "Build and deployment," change the "Source" from "Deploy from a branch" to "GitHub Actions".

## Other change from previous stage :

- Adding CI action for github :
    - Adding linting using Ruff
    - Running test using pytest
    - Generate test report using coverage and pusblish on github pages available at [github pages test report](https://draed.github.io/devops_learning/test-report/index.html)

## Usage

- make change under any folder and git push action will trigger the CI job
