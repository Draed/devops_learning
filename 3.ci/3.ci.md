# 3.ci

*Created on:* 2025-12-23

## Description 

Adding CI (Continuous Integration)
- Adding github action workflow

## Prerequisites

- Have python 3 installed
- Enabling github pages in repository setting -> pages -> GitHub Pages -> Build and deployment section -> Branch :
    - configure branch (default : main) 
    - configure folder (default : root)

## Other change from previous stage :

- Adding CI action for github :
    - Adding linting using Ruff
    - Running test using pytest
    - Generate test report using coverage and pusblish on github pages available at [github pages test report](https://draed.github.io/devops_learning/index.html)

## Usage

- make change under specific folder `3.ci` and git push action will trigger the CI job
